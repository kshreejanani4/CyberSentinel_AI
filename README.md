# CyberSentinel_AI
CyberSentinel AI : DDoS Intrusion Detection System using Machine Learning for network traffic classification (BENIGN vs ATTACK)
AQI Forecast System

Project Overview  
This project predicts Air Quality Index (AQI) values using machine learning techniques. The dataset is cleaned, analyzed, visualized, and used to train a regression model for prediction.

Technologies Used  
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Google Colab  

Project Workflow  
1. Data cleaning  
2. Missing value handling  
3. Data visualization  
4. Correlation analysis  
5. Feature selection  
6. Model training  
7. Prediction and evaluation  

Model Used  
Random Forest Regressor is used for AQI prediction. It is suitable for regression problems and handles non-linear relationships in data effectively.

Model Performance  
- R2 Score: 0.646  
- Mean Absolute Error: 22.40

- Dataset preview

- Actual vs predicted graph
